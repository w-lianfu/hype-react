# Hype-React

## 可用命令

### `npm start`
* 启动项目

### `npm run build`
* 构建项目

### `npm run lint`
* 启动 eslint 检查

---

## 关于框架
* 使用的主要技术有: React + TypeScript + React-Router + Mobx + Material-UI + Sass.
* 另外已经安装的 package 有: Rxjs, Immutable, Moment, Ramda, Lodash, Emotion, Viewerjs, Mathjs.
* 项目图标使用 material-design-icons.
